# ERC20 Event Log

### How to use
1. Install [Node.js](https://nodejs.org/en/) and [npm](https://www.npmjs.com/)
2. Install npm dependencies
```bash
npm install
```
3. Copy .env.example to .env and fill in the values
```bash
cp .env.example .env
```

4. Edit .env file
```properties
# .env
PROVIDER_URL=
PRIVATE_KEY=
```

5. Run the script
```bash
node index.js
```
